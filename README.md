# Vben-admin

## 介绍

该项目借鉴于开源项目[vue-vben-admin](https://github.com/vbenjs/vue-vben-admin)v5.5.7进行二次开发，作用在于：

1. 对接后端的后台管理系统
2. 快速迭代开发
3. 沉淀已有的技术实践

自主部署的vue-vben-admin[文档地址](https://vben-admin-docs.vercel.app/)，这是为了防止该开源项目随时间更新后与当前版本文档不一致的问题

> **项目瘦身指南**参考：https://vben-admin-docs.vercel.app/guide/introduction/thin.html
